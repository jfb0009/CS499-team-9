Course Functions

Refer to the following for functionality regarding courses:

  • Courses can be created by clicking the Add Course button and can be removed by clicking the Remove Course

  • By selecting Add Term or Remove Term, professors can add/remove the term.
  • Professors can make as many courses as needed for their current semester.
  • By selecting Reset Password, professors may reset there account password.
  • Simply select Logout to exit the account.

    Assignment Functions

    The assignment functionality is focused on adding assignment and utilizing them to look up grades and for calclations:
    The calculations include mean, standard deviation, and overall grade.

    • Professors can add/remove assignments by clicking the Add Assignment or Remove Assignment buttons respectively.

    • Add or Remove catergories for assignments by using the Add Category or Remove Category buttons.
    • Grades can be added for assignments.
    • Professors can also search for grades via the assignment name.

    Grade Functions

    Functions include:

    • Professors can create grading scales and import past grading scales for courses.

    • Calculating Final Grade
    • Calculating Mean for Class
    • Calculating Standard Deviation for the Class

    Student Functions

    Functions include:

    • Professors can add or remove students from course(s).
    • Print student rosters.
    • Search via student name.
